# TXT Merger - 複数のテキストファイルを1つにまとめるツール

ブラウザだけで複数の `.txt` ファイルをドラッグ＆ドロップして、**1つのテキストファイルにまとめてダウンロード**できる超シンプルなツールです。  
インストール不要で動作します。
## 特徴

- 完全オフライン（個人情報もファイルも一切送信されません）
- ドラッグ＆ドロップ or クリックでファイル選択
- 結合順は選択した順番そのまま
- 各ファイルの末尾に自動で改行を追加（重複改行を防ぐ簡単処理付き）
- UTF-8対応で日本語も文字化けしません
- 軽量・高速・広告なし

## 使い方

1. このリポジトリをダウンロード or Clone
2. `txt-merger.html` をダブルクリックでブラウザで開く
3. TXTファイルをドロップするかクリックして選択
4. 「結合してダウンロード」ボタンを押す → 完成！

→ ダウンロードされるファイル名例：`結合結果_2025-11-26T15-30-45.txt`
